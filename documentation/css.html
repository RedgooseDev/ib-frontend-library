시작하기

이 CSS 라이브러리는 i-builder로 제작하는 사이트를 스타일을 규격화 시키기 위하여 만들어 졌습니다.
이것은 Bootstrap front-end framework를 참고하여 제작하였습니다.

지원하는 브라우저의 최저버전은 ie8입니다. 모든 브라우저가 동일한 형태로 출력하지 않으며 브라우저 버전에 따라 점진적으로 나아지는 형태고 개발하는것으로 목표하고 있습니다.

Headding

제목에 관한 스타일을 정의합니다.
div.h1 ~ div.h6와 같이 특정 엘리먼트에 클래스를 붙여 사용할 수 있습니다.
headding 엘리먼트 내부에 <small/>, <span.small/> 엘리먼트를 사용하면 좀 더 작은 사이즈의 글자를 표현할 수 있습니다.
